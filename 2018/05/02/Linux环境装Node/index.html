Linux环境装Node

在windows的环境上装node 是非常容易的一件事情，只需要找到相应的资源，点击安装，下一步下一步就行了，Linux环境下没有了图形化的使用界面，那么我们应该怎样来安装Node了。

第一步 首先还是找到相应的资源

Node.js官网 http://nodejs.cn/download/

如果你不知道自己应该装什么位数的系统了，在Linux上使用getconf LONG_BIT 命令来查看。


如图，我的是64位。

第二步：对应的资源传到服务器上

从Windows传到Linux的工具有很多，我使用这一款winScp,简洁大方款。将文件上传到指定的文件夹后，就要进行解压的操作了，进入指定的文件夹，运行tar -xvf ‘你上传的node打包文件夹’ 进行解压，解压后会有一个同名的nodejs文件，这时候利用mv 进行重命名


如图 利用mv将文件夹重命名为nodejs

第三步：设置环境变量

这里要将node设为Linux的全局变量，这里使用Linux的软连接来进行设置。主要利用 ln指令来建立软连接


如图 利用ln -s 将node文件夹中的npm和node 与bin文件夹建立了联系

第四步：检测是否安装node

这一步很简单，就是使用node-v npm-v 来使用就行。

当你看到这个的时候，表示你已经安装成功了。
